# 🚀 Рекомендации по улучшению приложения

## 📊 Анализ текущего функционала

### ✅ Что уже реализовано:
- ✅ Аутентификация (регистрация, вход, смена пароля)
- ✅ Чаты (создание, добавление участников, удаление)
- ✅ Сообщения (отправка, получение, удаление, очистка чата)
- ✅ WebSocket для real-time обновлений
- ✅ Загрузка изображений в Яндекс Object Storage
- ✅ Сжатие изображений с сохранением оригинала
- ✅ Учет занятий (студенты, занятия, транзакции)
- ✅ Отчеты с автоматическим парсингом
- ✅ Выписки по счету
- ✅ Пагинация сообщений
- ✅ Оптимизация памяти для изображений
- ✅ Push-уведомления о новых сообщениях (FCM для iOS/Android, переход в чат по тапу, сброс токена при выходе)
- ✅ **Поиск по сообщениям** — поисковая строка в чате, API `GET /messages/chat/:chatId/search`, подсветка и переход к найденному
- ✅ **Статусы прочитано** — `read_at`, `message_reads`, mark as read при открытии чата (POST read-all)
- ✅ **Редактирование сообщений** — только свои, метка «отредактировано», `edited_at`, WebSocket
- ✅ **Ответы на сообщения (Reply/Quote)** — кнопка «Ответить», цитата, связь с оригиналом
- ✅ **Голосовые сообщения** — запись, загрузка в Object Storage, воспроизведение в чате
- ✅ **Пересылка сообщений** — «Переслать», выбор чата, контекст (от кого, когда)
- ✅ **Закреплённые сообщения** — pin/unpin, блок до 3 закреплённых в шапке чата, до 5 на чат
- ✅ **Реакции на сообщения (эмодзи)** — долгое нажатие → выбор эмодзи, отображение под сообщением, WebSocket
- ✅ **Тёмная и светлая тема** — «Ультрафиолет (тёмная)» / «Аврора (светлая)», переключатель в профиле, сохранение выбора
- ✅ **Голосовые звонки 1-на-1** — WebRTC (`VoiceCallService`), сигнализация по WebSocket, coturn, push для входящих
- ✅ **Видео в чате** — выбор/запись видео из галереи или камеры, отправка как файл
- ✅ **Экспорт чата** — меню чата → «Экспорт чата» → TXT-файл
- ✅ **Папки чатов** — до 5 пользовательских папок, фильтр в списке чатов
- ✅ **Профили пользователей** — экран с аватаром и email (`UserProfileScreen`)
- ✅ **E2EE (базовый)** — шифрование сообщений и медиа в чатах (не отдельный режим «секретный чат»)
- ✅ **Drag-and-drop** — перетаскивание файлов в чат на desktop
- ✅ **Кэширование изображений** — `cached_network_image`, дисковый кэш, memCacheWidth, placeholder
- ✅ **Офлайн** — кэш сообщений (Hive), очередь отправки, retry при ошибке, индикатор «В очереди»
- ✅ **Жесты в чате** — Pull to refresh (обновление списка сообщений), свайп по сообщению влево: «Удалить» (свои), вправо: «Ответить»; долгое нажатие открывает меню (ответить, переслать, реакция, редактировать, закрепить, удалить)
- ✅ **Анимации и переходы** — плавный переход при навигации (слайд справа + затухание, экран чата и профиля); анимация появления сообщения в чате (fade + лёгкий scale); skeleton loading при загрузке изображений (пульсирующий плейсхолдер в сообщениях, превью ссылок, галерее)

### UX/UI

#### 22. ~~Анимации и переходы~~ ✅ Реализовано
- **Навигация** — [slideAndFadeRoute](../lib/utils/page_routes.dart): слайд справа + затухание при открытии чата и профиля.
- **Появление сообщения** — [FadeScaleIn](../lib/widgets/fade_scale_in.dart): однократная анимация (opacity + scale) при появлении сообщения в списке.

#### 23. ~~Жесты~~ ✅ Реализовано
- **Pull to refresh** — список сообщений в чате обновляется по потягиванию вниз.
- **Свайп по сообщению** — вправо: «Ответить», влево (только свои): «Удалить» (Slidable).
- **Долгое нажатие** — открывает меню сообщения (ответить, переслать, реакция, редактировать, закрепить, удалить и др.).

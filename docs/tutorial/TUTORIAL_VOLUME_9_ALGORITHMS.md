## 1. Поиск в списке

- **Первый элемент по условию:** `list.firstWhere((e) => e.id == id, orElse: () => throw StateError('Not found'));` или `list.cast<Chat?>().firstWhere((e) => e?.id == id, orElse: () => null);`.
- **Есть ли элемент:** `list.any((e) => e.id == id)`.
- **Индекс элемента:** `list.indexWhere((e) => e.id == id)` (или -1).
- **Линейный поиск** — перебор for/for-in до первого совпадения; для больших списков при частых поисках可以考虑 использовать Map по ключу (id → объект).

В проекте: поиск чата по id в списке, проверка «есть ли пользователь в участниках».

## 2. Сортировка списка

- **list.sort((a, b) => a.field.compareTo(b.field));** — по полю (например, по имени). Для чисел: `a.compareTo(b)` или `a - b`.
- **Обратный порядок:** `list.sort((a, b) => b.field.compareTo(a.field));` или после sort вызвать `list.reversed.toList()`.
- **По нескольким полям:** в компараторе сначала сравнить первое поле, при равенстве — второе.
- **Создать новый список, не меняя исходный:** `final sorted = List.from(list)..sort(...);`

В проекте: сортировка чатов по дате последнего сообщения, порядок по сохранённому _chatOrder.

## 3. Set: множество уникальных элементов

**Set** — коллекция без дубликатов. Удобно хранить идентификаторы выбранных элементов (например, выбранные userId при создании чата).

```dart
final Set<String> ids = {};
ids.add('1');
ids.add('1');  // дубликат не добавится
ids.contains('1');  // true
ids.remove('1');
```

В проекте: _selectedUserIds в диалоге создания чата (HomeScreen) — Set<String>.

## 4. Фильтрация и преобразование

- **where:** `list.where((e) => e.isActive).toList()`.
- **map:** `list.map((e) => e.name).toList()`.
- **where + map:** отфильтровать и преобразовать в один проход.

Пример: список чатов отфильтровать по папке и по поисковому запросу (в HomeScreen так строится filteredChats).

---

## 5. Группировка и подсчёт

- **Группировка по ключу:** вручную — `Map<String, List<Chat>> groups = {}; for (final c in chats) { groups.putIfAbsent(c.folderId ?? '', () => []).add(c); }`.
- **Подсчёт:** `list.where((e) => e.unreadCount > 0).length` или цикл с счётчиком.

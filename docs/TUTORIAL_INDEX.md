# Полный учебник по программированию: Dart и Flutter (план на ~1000 страниц)

Это **план расширенного учебника**, по которому можно пройти путь от нуля до уверенного чтения и изменения кода приложения-чата (my_chat_app). Учебник разбит на **10 томов**; все тома лежат в папке **docs/tutorial/**.

**Как пользоваться:** читайте тома по порядку (01 → 02 → … → 10). В конце каждого тома указано, какой том идёт следующим.

---

## Где лежат тома

**Все тома учебника находятся в папке [docs/tutorial/](tutorial/).**

- Оглавление и ссылки на тома — ниже в этом файле.
- В папке `tutorial/` также есть [README.md](tutorial/README.md) с кратким описанием.

---

## Связь с основным учебником

- **[DART_FLUTTER_TUTORIAL.md](DART_FLUTTER_TUTORIAL.md)** — краткий учебник «на один проход»: все темы по верхам, с примерами из проекта.
- **Тома в docs/tutorial/** — развёрнутый вариант: больше объяснений, пошаговые примеры, разбор типичных ошибок, задачи и проверь себя. Рассчитаны на обучение программированию с нуля.
